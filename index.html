OpenAPI SDK Generator Comparison

Interactive comparison of four TypeScript OpenAPI SDK approaches: openapi-zod-client + Zodios, @hey-api/openapi-ts + client-fetch, @hey-api/openapi-ts + Zod, and openapi-typescript + openapi-fetch

openapi-zod-client + Zodios

Runtime validation with Zod schemas and type-safe API client

@hey-api/openapi-ts + client-fetch

Modern fetch-based client with TypeScript types

@hey-api/openapi-ts + Zod

Hey API client with Zod validation for best of both worlds

openapi-typescript + openapi-fetch

Minimal runtime with strong TypeScript inference
